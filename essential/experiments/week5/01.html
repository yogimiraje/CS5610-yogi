About this experiment

This page explains the ng-routing feature of Angular JS. The file petition page provides a form to fill the petition data. View petitions display this data. With ng-routing, you can create a multi-page application without any request going back to server. The first experiment is setting up angular js enviornment with different ng directives.

Code snippet

ng-routing: configuration

              app.config(['$routeProvider',
                  function ($routeProvider) {
                      $routeProvider.
                        when('/home', {
                            templateUrl: 'partials/home.html'
    
                        }).
        
                         when('/file', {
                             templateUrl: 'partials/file.html',
                             controller: 'FileController'
                         }).
                        when('/view', {
                            templateUrl: 'partials/view.html',
                            controller: 'ViewController'
                        }).
                         when('/contact', {
                             templateUrl: 'partials/contact.html',
                         }).
                        otherwise({
                            redirectTo: '/home'
                        });
                  }]);
