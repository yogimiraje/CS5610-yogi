DOM visibility using Jquery

Click on any of the images to make it disappear. You can click on show all button to make all images reappear.





About this experiment

This experiment illustartes the ways you can manipulate the DOM's visibility. This is achieved by hide() and show() methods. A click on particular image is grabbed with image's id and hide method is applied to it.

Code snippet

jQuery


            $("#empire").click(function () {
            $("#empire").hide();
             });
    
    
            $("#showAll").click(function () {
                $("#empire").show();
        
            });
