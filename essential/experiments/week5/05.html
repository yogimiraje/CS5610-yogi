About this experiment

This experiment shows the changing of the classes using jQuery. With jQuery, you can manipulate any DOM's class. This is achieved using addClass() and removeClass() methods.

Code snippet

jQuery:


         $("#blueButton").click(function () {
         
            $("#sample").removeClass("red");
            $("#sample").addClass("blue");
            
        });
        $("#redButton").click(function () {
            $("#sample").removeClass("blue");
            $("#sample").addClass("red");
        });
