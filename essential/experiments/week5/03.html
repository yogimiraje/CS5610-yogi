About this experiment

This page explains the ng-routing feature of Angular JS. The file petition page provides a form to fill the petition data. View petitions display this data. With ng-routing, you can create a multi-page application without any request going back to server. The first experiment is setting up angular js enviornment with different ng directives. In second experiment, I created diffrent pages and set up the navigation through those pages. This is achieved using creating different 'tabs' (partial pages). You may put different contents for diffrent pages based on the page is made for. In third experiment, I set up a service that enables communication between the pages. FileController is the controller that manages filed petitions. This data is passed to 'PetitionService' , so that it can be accessed by all the controllers that needs the petitions data.

ng-routing: configuration

              app.config(['$routeProvider',
                  function ($routeProvider) {
                      $routeProvider.
                        when('/home', {
                            templateUrl: 'partials/home.html'
    
                        }).
        
                         when('/file', {
                             templateUrl: 'partials/file.html',
                             controller: 'FileController'
                         }).
                        when('/view', {
                            templateUrl: 'partials/view.html',
                            controller: 'ViewController'
                        }).
                         when('/contact', {
                             templateUrl: 'partials/contact.html',
                         }).
                        otherwise({
                            redirectTo: '/home'
                        });
                  }]);

                };

ng-routing: service

                  app.factory('PetitionService', function PetitionService($http) {
                var myPetitions = [];
                console.log("In service");
                var addToPetitions = function (petition) {
                    myPetitions.push(petition);
        
                    console.log("Added - display from service");
                    console.log(petition);
                }
   
                var getPetitions = function () {
                    console.log("returning petition");
                    return myPetitions;
                }
                return {
                    addToPetitions: addToPetitions,
                    getPetitions: getPetitions
                }
                                                                                                        
            });
