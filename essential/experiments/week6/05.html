About this experiment

In these experiments, I have extended the functionality of experiment I performed last week. Main goal was to give profile-based access to user. The user still can file the petition without signing in. However, after signing in you can see the petitions you have filed. Let us see how this is achieved in a series of experiments.

In first experiment, I created different sample profiles. You may try login using :
username: joe, lucy,matt with passwords same as that of username. After login, you will see some additional links and also a logout button.

In second experiment, I added the logout functionality. After logging out , you will again see the login form again. 'profileController handles the login , logout functionality, although all user data resides in the UserService, so that all pages can get the information about the currently logged in user. We also need to make sure that login form is reset so that login information of the last user is reset.

In third experiment , I added the login validation feature. so, if you try to login with invalid username or password, an error alert is thrown.

As part of fourth experiment , I updated the user's petitition, so that user can see his own petitions. This is done by adding the petitionsFiled array to the user data.

In fifth experiment , I made sure that if an user is logged in and files a petition, it is updated to that user's individual petitions as well. For this, I created addPetitionIndex function that updates the index of the petition filed to petitionsFiled array using 'petitionsFiled.push(index)' function.

ng-routing: service

 
                app.factory("UserService", function () {
                var currentUser = null;
                var users = [
                    { username: "joe", password: "joe", petitionsFiled: [1, 2] },
                    { username: "lucy", password: "lucy", petitionsFiled: [0, 3] },
                    { username: "matt", password: "matt", petitionsFiled: [4] }
                ]
                var login = function (username, password) {
                    for (var u in users) {
                        if (users[u].username == username && users[u].password == password) {
                            currentUser = users[u];
                            return users[u];
                        }
                    }
                    return null;
                };
                var getCurrentUser = function () {
                    return currentUser;
                }
                var addPetitionIndex = function (index, username) {
                    for (var u in users) {
                        if (users[u].username == username) {
                            users[u].petitionsFiled.push(index);
                        }
                    }
                };
                return {
                    login: login,
                    getCurrentUser: getCurrentUser,
                    addPetitionIndex: addPetitionIndex
                };
            });
