Registration of a new user

Experiment hosted on openshift

About this experiment

In this experiment, application allows the new user to register. User fills the registration form and user's data is passed to server, where it is added in the local array. So, you can try login with the newly added user name. On the server side user is parsed from the request body and added to local array using push method e.g. usersData.push(user).

Code snippet

server side js 

        app.post("/register", function (req, res) {
        var user = req.body;
        usersData.push(user);
        console.log("registered!");
        res.json(user);
        });
