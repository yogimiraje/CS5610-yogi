Static page display from Node.js

Experiment hosted on openshift

About this experiment

In this experiment, I am hosting a simple static page on openshift. This page is hard-coded static page which is residing on remote server. server.js is configured such that the changes can be tested both locally and remotely.

Code snippet

server.js 

            var express = require('express');
            var app = express();
            app.use(express.static(__dirname + '/public'));
            var ip = process.env.OPENSHIFT_NODEJS_IP || '127.0.0.1';
            var port = process.env.OPENSHIFT_NODEJS_PORT || 3000;
            app.listen(port, ip);
