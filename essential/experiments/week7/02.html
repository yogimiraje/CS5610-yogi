Simple Angular js application on the server using Node.js and express

Experiment hosted on openshift

About this experiment

In this experiment, I am hosting angular js application openshift. I created angular js application as usual and tested it locally. Once it ran fine locally, I commited the changes to git and pushed it to the remote server.

Code snippet

server.js 

            var express = require('express');
            var app = express();
            app.use(express.static(__dirname + '/public'));
            var ip = process.env.OPENSHIFT_NODEJS_IP || '127.0.0.1';
            var port = process.env.OPENSHIFT_NODEJS_PORT || 3000;
            app.listen(port, ip);
